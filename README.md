# Titanic EDA Analysis 🚢

## 📌 Objective
Analyze Titanic dataset to understand survival patterns.

## 📊 Key Analysis
- Survival count
- Survival by gender
- Survival by class
- Age distribution
- Correlation analysis

## 🧠 Insights
- Females had higher survival rates
- First-class passengers survived more
- Fare influenced survival chances

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
